## Architecture

### Frontend
- **Framework:** React 19.2.0 + Vite 7.3.1
- **Styling:** TailwindCSS
- **State:** Zustand
- **HTTP:** Axios
- **Other:** react-router-dom, @dnd-kit (drag & drop), lucide-react, react-to-print
- **Build command:** `npm run build`
- **Output dir:** `dist/`
- **Dev server port:** 5173

### Backend
- **Framework:** FastAPI + Uvicorn
- **Language:** Python
- **Root dir in repo:** `backend/`
- **Start command:** `uvicorn main:app --host 0.0.0.0 --port 10000`
- **Build command:** `pip install -r requirements.txt`
- **Key dependencies:** openai, fastapi, uvicorn, pydantic, python-dotenv, requests

### API Endpoints
| Method | Path | Description |
|--------|------|-------------|
| POST | /api/ai/score | Score resume against a job description (OpenAI) |
| POST | /api/ai/optimize | Optimize bullet points with AI |
| GET | /health | Health check → `{"status": "ok"}` |
| * | /api/payments/* | Payment processing |
